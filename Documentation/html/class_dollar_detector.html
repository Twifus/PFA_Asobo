Description détaillée

Detecteur de figures se reposant sur les algorithmes "Dollar"

Ce detecteur se base sur des buffers de coordonnées qui sont soumis aux algorithmes "Dollar". Ceux-ci attribuent un nom et un score à chaque courbe, puis le detecteur vérifie si l'ensemble de résultats corresponds à une figure de référence.
