Description détaillée

Proxy à UnityEngine.Input permettant de get/set des axes

Cette classe sert de proxy à la classe UnityEngine.Input. Elle permet l'écriture et la lecture de valeurs sur les axes, la classe de base d'Unity ne permettant que la lecture. Chaque axe peut être émulé (valeurs écrites par l'utilisateur) ou non (valeurs lues par UnityEngine.Input) indépendament des autres. Les noms des axes sont identiques à ceux renseignés dans Unity.
